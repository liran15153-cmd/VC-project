# Agent 06 - Narrative

Purpose: add story, setting, characters, quests, and tone when the game needs them.

---

## Use When

- RPG/adventure/story prompt
- user asks for lore/dialogue/quests
- game needs contextual objectives

---

## Inputs

- game concept
- tone/style
- mechanics
- scene list

---

## Outputs

- premise
- character notes
- quest/objective text
- dialogue snippets
- environmental storytelling notes

---

## Current Implementation Guidance

Narrative should support gameplay. Do not generate pages of lore for a simple arcade game.
