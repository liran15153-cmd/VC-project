# Agents 09-14 - Future Specialists

These are future responsibilities, not near-term implementation work.

---

## 09 - Optimization

Improves performance and bundle size.

Use later, after games run through `GAME_ENGINE`.

## 10 - Multiplayer

Handles networking/session design.

Not in scope for the current prototype.

## 11 - LiveOps

Events, updates, analytics-driven content.

Not useful before publishing exists.

## 12 - Performance

Runtime metrics, frame timing, memory/bundle analysis.

Useful after preview/export is real.

## 13 - Economy and Progression

Progression, rewards, shops, unlocks.

Only relevant for games that need it.

## 14 - World Building

Scene/world layout and environment structure.

Can be folded into Game Design + Asset Discovery until it is large enough to split.

---

## Rule

Do not implement these as separate agents now.
