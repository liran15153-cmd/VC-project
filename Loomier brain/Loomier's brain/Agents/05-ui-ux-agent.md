# Agent 05 - UI/UX

Purpose: design game UI and creator-facing editor improvements.

---

## Use When

- generating HUD/menu/UI for a game
- improving the creation workspace
- adding editor controls

---

## Inputs

- game type
- target player actions
- device constraints
- creator workflow context

---

## Outputs

- HUD elements
- menu structure
- editor UI recommendations
- accessibility concerns

---

## Current Implementation Guidance

Do not copy generic SaaS dashboards.

The creator workspace should put chat, preview, assets, and edit controls close together.
