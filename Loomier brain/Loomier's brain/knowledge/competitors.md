# Competitor Notes

This is a lightweight competitor memory, not a full market report.

---

## Relevant Categories

| Category | Examples | What To Learn |
| --- | --- | --- |
| AI app builders | Base44, Lovable, Bolt | Fast creation flow and clean UI |
| AI game generators | Rosebud, Astrocade, Layer | Prompt-to-play expectations |
| Game engines | Unity, Godot, Construct, GDevelop | Runtime/editor expectations |
| Asset ecosystems | Itch, Unity Asset Store, Kenney | Asset reuse and licensing |

---

## Our Differentiation

LOOMIER should win on:

- editable generated games
- asset-aware creation
- hybrid 2D/3D runtime
- fast iteration
- creator ownership

It should not try to win by copying generic SaaS dashboards.

---

## Product Warning

Base44-style polish is useful as inspiration, but copying that UX too literally can make the product feel like an app builder instead of a creative game tool.

The first screen should be the creation workspace, not a marketing dashboard.
