# Platform Vision

See [[docs/vision]] for the concise version.

---

## Long-Term Direction

LOOMIER should become a place where creators can:

- prototype game ideas quickly
- reuse and remix assets
- collaborate with AI
- visually edit generated worlds
- publish playable experiences

---

## Strategic Bet

The winning feature is not "AI can generate code."

The winning feature is:

```txt
AI can help maintain an editable game project over time.
```

That requires structured data, assets, runtime systems, and validation.
